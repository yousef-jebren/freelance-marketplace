# Freelance Marketplace Website

## Description
A full-stack web application that allows freelancers to create and manage services, and clients to browse and purchase them.

## 🚀 Features
- User authentication (Freelancer & Client roles)
- Create and manage services
- Browse and purchase services
- Database integration (MySQL)

## 🛠️ Technologies Used
- HTML
- CSS
- PHP
- MySQL
- WAMP Server
